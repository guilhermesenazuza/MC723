# Relatório Projeto 1: Benchmarking

##### Pedro Rodriques Grijó
##### 139715


### Introdução

Este relatório tem por objetivo comparar o desempenho dos computadores utilizados nas partes 1 e 2 do projeto 1 da disciplina MC723 - Laboratório de Projetos de Sistemas Computacionais. Na parte 1, nos separamos em grupos para definição de um benchmark utilizando um programa escolhido pelo grupo. Na parte 2, os grupos foram re-formados, e cada programa da parte 1 teve seu desempenho medido em pelo menos 3 novos computadores. Aqui serão feitas considerações sobre o desempenho das máquinas para cada um dos 8 benchmarks.
